# bscthesis
BSc Thesis about wildfires

## Notebooks:
The notebooks are all named systematically, and have a combination of a number and a letter. Here you can find an overview of the meaning of the characters.  

a - data from PhiSat-2  
b - data from segTHRawS  
c - data from THRawS  
00 - initial look at data  
01 - Preprocessing/saving data
03 - Baseline model U-Net  
04 - Evaluation of models  

clf - classification model baselines
seg - segmentation model baselines
